## 对比方法

### random

随机生成查询，然后从中选择最好的结果

### feedback_based

使用基于反馈的方法生成查询，使用概率表随机生成查询，然后根据feedback来调优概率表。之前有数据库测试的论文是这么搞的。

### genetic

使用遗传算法生成查询，每次生成一批查询，然后根据结果进行演化变异的步骤，最终统计结果
